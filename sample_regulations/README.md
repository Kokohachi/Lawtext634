## Lawtext形式について

Lawtext形式は、日本の法令や規約を人間が読みやすく、かつ編集しやすいプレーンテキスト形式です。

### 基本構造

```
規約名
（令和X年規約第Y号）

      第一章　章タイトル

  （見出し）
第一条　条文の内容…
２　第二項の内容…
```

### 特徴
- プレーンテキストで管理可能
- Gitでのバージョン管理に適している
- 法令標準XMLとの相互変換が可能
- 既存のテキストエディタで編集可能
